Ao invés de ser obrigado a passar horas a fio formatando tabelas em Excel softwares pagos, você pode usar pacotes gratuitos do R para formatar automaticamente suas tabelas:

  • Estatísticas Descritivas: gtsummary::tbl_summary()
  • Correlações: sjPlot::tab_corr()
  • Regressões: sjPlot::tab_model()

Estatísticas Descritivas

O pacote gtsummary possui um conjunto de funções para sumarizar dados e tabelas. Eu particularmente gosto da função gtsummary::tbl_summary(). Ela formata uma tabela de Estatística Descritiva de maneira bem conveniente.

gtsummary::tbl_summary(
  kidiq,
  by = mom_hs,
  type = all_continuous() ~ "continuous2",
  statistic = list(
    all_continuous() ~ c("{N_nonmiss}",
                         "{median} ({p25}, {p75})", 
                         "{min}, {max}"),
    all_categorical() ~ "{n} ({p}%)"),
  missing = "no",
  digits = all_continuous() ~ 2) %>%
  # add p value and overall
  add_p(pvalue_fun = ~style_pvalue(.x, digits = 2)) %>% 
  add_overall() %>%
  # bold variable labels, italicize levels
  bold_labels() %>%
  italicize_levels() %>%
  # change stuff
  modify_header(label ~ "**Variable**") %>% 
  modify_spanning_header(c("stat_1", "stat_2") ~ "**Mom High School**") %>% 
  add_n()

Tabela de Correlações

Para as tabelas de correlações, eu uso o pacote sjPlot com a função sjPlot::tab_cor()

Os astericos significam:

  • * - \(p < 0.05\)
  • ** - \(p < 0.01\)
  • *** - \(p < 0.001\)
sjPlot::tab_corr(
  kidiq %>% mutate(mom_hs = as.integer(mom_hs)),
  digits = 2,
  triangle = "lower"
)

Modelo de Regressão Binomial/Logística

Vamos utilizar o caso da Aula 6 - Regressão Binomial

model_binomial <- stan_glm(
  switch ~ dist + arsenic + assoc + educ,
  data = wells,
  family = binomial()
)

Tabela de Regressão Binomial/Logística

A função sjPlot::tab_model() quando aplicada à um modelo bayesiano linear generalizado (binomial, Poisson etc.) já faz a transformação necessária para uma melhor interpretação dos coeficientes.

No caso de modelos binomiais/logísticos geralmente é aplicada uma exponenciação (exp()) dos coeficientes para transformá-los em razões de probabilidades (odds ratio)

Caso queira deixar os coeficientes brutos (raw coefficients) use transform = NULL
